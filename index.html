Instructions

  • Import a image of the moire disk design into Image #1 by selecting the desired file section.
  • If Disk 2 shares the same design as Disk 1, check the "Same Design" checkbox. If Disk 2's design is flipped, check the "Flipped" checkbox.
  • Adjust the background color by clicking on the gray box near the text "BG Color."
  • You can also animate the background color by clicking on blue button, "BG Color" and selecting to "Animated BG Color".
  • For the Animated BG Color, just hex codes for colors and seperate them with commas.
  • Initiate or pause the rotation by clicking the "Start" button.
  • If any issues occur with the rotation, such as being out of sync, press the "Reset Rotation" button to fix it.
  • Modify the rotation direction of each individual disk as needed by toggling the "Flip Direction" checkbox for each image.
  • Alter the rotation speed for each image to your preference (Higher values result in slower rotation).
